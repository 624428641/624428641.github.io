JS中Date对象

用来表示时间

该代码会直接封装为当前代码执行时间，如果创建一个指定日期的时间对象，需要在构造函数中传递一个表示时间的字符串（月/日/年 时：分：秒）作为参数
方法：
.getDate()获取当前对象日期的日
.getMonth()获取当前对象日期的月（0表示1月）
.getFullYear()获取当前对象日期的四位数年
.getDay()获取当前对象日期的星期（0表示周日）
.getTime()1970年1月1日（非中国时间，有八小时时差）至当前对象日期的毫秒数
计算机底层在保存时间时使用的都是毫秒（时间戳）
Date.now()获取当前时间戳

利用时间戳可以来测试代码的执行性能

.setDate .setMonth .setFullYear设置当前对象日期中的部分

DOM文档对象模型

document.querySelector

element = document.querySelector(selectors);

参数selectors：
包含一个或多个要匹配的选择器的 DOM字符串 DOMString。 该字符串必须是有效的CSS选择器字符串；如果不是，则引发SYNTAX_ERR异常。 使用反斜杠字符转义不属于标准CSS语法的字符。JavaScript使用退格转义，反斜杠是转义字符，所以当输入一个文本串时，必须将它转义两次（一次是为 JavaScript 字符串转义，另一次是为 querySelector 转义）：
返回值：
表示文档中与指定的一组CSS选择器匹配的第一个元素的 html元素Element对象。
如果需要与指定选择器匹配的所有元素的列表，则应该使用querySelectorAll() 。
异常SYNTAX_ERR：
指定selectors的语法无效。

如果选择器是一个 ID，并且这个 ID 在文档中错误地使用了多次，那么返回第一个匹配该 ID 的元素。
CSS 伪类不会返回任何元素

查找第一个匹配 class属性的html元素
var el = document.querySelector(".myclass");

查找一个class属性为”user-panel main”的div(<div class="user-panel main">)内包含一个name属性为”login”的input (<input name="login"/>)
var el = document.querySelector("div.user-panel.main input[name='login']");

高阶函数

函数指向某个变量，函数的参数接收变量，高阶函数就是一个函数就收另一个函数作为参数

简单高阶函数

Array数组中的高阶函数

  1. map()方法传入自己写的函数本身作为参数，得到了新的Array，可以有3个参数：callback(currentValue, index, array)
  2. reduce（）把一个函数作用在数组上，这个函数必须接收两个参数，把结果继续和下一个元素做计算
  3. filter()用于把某些元素过滤，返回剩下元素，接收一个函数作为参数，返回值是TRUE保留元素。还可以接收另外两个参数（元素的位置，数组本身）
  4. indexOf()方法返回在数组中可以找到一个给定元素的第一个索引，如果不存在，则返回-1
  5. sort()方法是用于排序的，字符串根据ASCII码进行排序，默认把所有元素先转换为String再排序。可以接收一个比较函数来实现自定义的排序，会直接对Array进行修改
  6. every()方法可以判断数组的所有元素是否满足测试条件
  7. find()方法用于查找符合条件的第一个元素，如果找到了，返回这个元素，否则，返回undefined
  8. findIndex()查找符合条件的第一个元素,返回这个元素的索引，如果没有找到，返回-1
  9. forEach()把每个元素依次作用于传入的函数，但不会返回新的数组。常用于遍历数组，传入的函数不需要返回值

闭包

函数和声明该函数的词法环境的组合

闭包是由函数以及创建该函数的词法环境组合而成。这个环境包含了这个闭包创建时所能访问的所有局部变量

返回的函数引用了变量i，但它并非立刻执行。等到3个函数都返回时，它们所引用的变量i已经变成了4
返回函数不要引用任何循环变量，或者后续会发生变化的变量
闭包就是携带状态的函数，并且它的状态可以完全对外隐藏起来。

箭头函数

x => x * x 相当于 function (x) { return x * x;}
箭头函数相当于匿名函数，并且简化了函数定义，箭头函数内部的this是词法作用域，由上下文确定
用call()或者apply()调用箭头函数时，无法对this进行绑定，即传入的第一个参数被忽略

Array对象

用于在单个的变量中存储多个值

如果参数只有一个为其长度，多个为值，没有参数的数组为空长度0
方法：
不改变原数组：

  • concat() 连接两个或多个数组
  • reverse() 方法用于颠倒数组中元素的顺序
  • join() 方法用于把数组中的所有元素放入一个字符串，返回一个字符串。
  • slice() 方法可从已有的数组中返回选定的元素，又两个参数，开始结束地方

改变原数组：

  • pop() 方法用于删除并返回数组的最后一个元素，数组长度减 1，返回它删除的元素的值。如果数组已经为空，则 pop() 不改变数组，并返回undefined。
  • push() 方法可向数组的末尾添加一个或多个元素，并返回新的长度，直接修改 arrayObject
  • shift() 方法用于把数组的第一个元素从其中删除，并返回第一个元素的值
  • splice() 方法对数组惊醒添加或者删除项目操作，然后返回被删除的项目
  • unshift() 方法可向数组的开头添加一个或更多元素，并返回新的长度

Math.ceil()向上取整

CSS伪类

:nth-child伪类

首先找到所有当前元素的兄弟元素，然后按照位置先后顺序从1开始排序，选择的结果为第（an+b）个元素的集合

2n+1匹配位置为1、3、5、7…的元素。可以使用关键字odd来替换此表达式
2n+0或简单的2n匹配位置为2、4、6、8…的元素。可以使用关键字even来替换此表达式。

:hover伪类

适用于用户使用指示设备虚指一个元素（没有激活它）的情况。这个样式会被任何与链接相关的伪类重写。可以任何伪元素上使用。会在光标（鼠标指针）悬停在元素上时提供关联的样式。

为了确保生效，按照LVHA的循顺序声明:link :visited :hover :active

:focus伪类

focus表示获得焦点的元素（如表单输入）。当用户点击或触摸元素或通过键盘的 “tab” 键选择它时会被触发。

此伪类仅适用于焦点元素本身。如果要选择包含焦点元素的元素，请使用:focus-within 。

AJAX

异步的JS和XML，是一种技术

客户端发起请求，服务器端处理响应，之后客户端页面载入=>同步
客户端发起请求，服务器端处理响应，同时客户端页面载入=>异步

new XMLHttpRequest()
HTTP请求
无状态协议，不建立持久的连接
组成：方法或动作，URL，头（客户端环境，身份验证），体（头和体中间有空行）

GET 用于信息查询获取，信息对任何人可见，数量限制在2000个字符，幂等
POST 用于新建删除修改，发送表单数据，无数量限制，
HTTP响应组成：状态码（数字文字显示是否成功），头（服务器信息），体

状态码：1XX正在处理，2XX成功，3XX重定向，没成功，4XX客户端错误，5XX服务器错误

发送请求方法：
open(method,url,async) async判断异步
setRequestHeader() 设置头信息，当method为post写在中间
send(string)

取得响应：
reponseText 字符串形式
reponseXML XML形式
status/statusText 返回状态码
getAllResponseHeader 获取所有响应报头
getReaponseHeader 某个字段的值
readyState属性

属性显示 作用
0 open未调用
1 连接已建立
2 请求已接收
3 请求处理中
4 响应完成
