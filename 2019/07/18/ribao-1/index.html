高阶函数

函数指向某个变量，函数的参数接收变量，高阶函数就是一个函数就收另一个函数作为参数

简单高阶函数

Array数组中的高阶函数

  1. map()方法传入自己写的函数本身作为参数，得到了新的Array，可以有3个参数：callback(currentValue, index, array)
  2. reduce（）把一个函数作用在数组上，这个函数必须接收两个参数，把结果继续和下一个元素做计算
  3. filter()用于把某些元素过滤，返回剩下元素，接收一个函数作为参数，返回值是TRUE保留元素。还可以接收另外两个参数（元素的位置，数组本身）
  4. indexOf()方法返回在数组中可以找到一个给定元素的第一个索引，如果不存在，则返回-1
  5. sort()方法是用于排序的，字符串根据ASCII码进行排序，默认把所有元素先转换为String再排序。可以接收一个比较函数来实现自定义的排序，会直接对Array进行修改
  6. every()方法可以判断数组的所有元素是否满足测试条件
  7. find()方法用于查找符合条件的第一个元素，如果找到了，返回这个元素，否则，返回undefined
  8. findIndex()查找符合条件的第一个元素,返回这个元素的索引，如果没有找到，返回-1
  9. forEach()把每个元素依次作用于传入的函数，但不会返回新的数组。常用于遍历数组，传入的函数不需要返回值

闭包

函数和声明该函数的词法环境的组合

闭包是由函数以及创建该函数的词法环境组合而成。这个环境包含了这个闭包创建时所能访问的所有局部变量

返回的函数引用了变量i，但它并非立刻执行。等到3个函数都返回时，它们所引用的变量i已经变成了4
返回函数不要引用任何循环变量，或者后续会发生变化的变量
闭包就是携带状态的函数，并且它的状态可以完全对外隐藏起来。

箭头函数

x => x * x 相当于 function (x) { return x * x;}
箭头函数相当于匿名函数，并且简化了函数定义，箭头函数内部的this是词法作用域，由上下文确定
用call()或者apply()调用箭头函数时，无法对this进行绑定，即传入的第一个参数被忽略

Array对象

用于在单个的变量中存储多个值

如果参数只有一个为其长度，多个为值，没有参数的数组为空长度0
方法：
不改变原数组：

  • concat() 连接两个或多个数组
  • reverse() 方法用于颠倒数组中元素的顺序
  • join() 方法用于把数组中的所有元素放入一个字符串，返回一个字符串。
  • slice() 方法可从已有的数组中返回选定的元素，又两个参数，开始结束地方

改变原数组：

  • pop() 方法用于删除并返回数组的最后一个元素，数组长度减 1，返回它删除的元素的值。如果数组已经为空，则 pop() 不改变数组，并返回undefined。
  • push() 方法可向数组的末尾添加一个或多个元素，并返回新的长度，直接修改 arrayObject
  • shift() 方法用于把数组的第一个元素从其中删除，并返回第一个元素的值
  • splice() 方法对数组惊醒添加或者删除项目操作，然后返回被删除的项目
  • unshift() 方法可向数组的开头添加一个或更多元素，并返回新的长度
