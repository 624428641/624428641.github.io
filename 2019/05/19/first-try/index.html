  • 水平垂直居中

    用相对绝对定位和负边距实现上下左右居中：高度和宽度已知

    利用绝对定位来实现居中：适合高度，宽度已知的情况。

    将父盒子设置为table-cell元素，text-align:center和vertical-align:middle实现水平垂直居中,父盒子高宽已知

    使用css3中的transform来时实现水平垂直居中：适合高度宽度未知的情况

    使用Flexbox来实现水平垂直居中,设置是相对于所有子元素进行计算的

  • 盒模型

    盒模型包含：margin、padding、border、内容区域
    嵌套结构下，当父级没有明确的上边界时(border，padding)，这时候子级设置margin-top的话，就会传递给父级，带着父级一起向下移动。
    整体的宽高就是 开始设置的宽高+padding+border
    设置了box-sizing:border-box;后，盒子变成了怪异盒模型。
    怪异盒模型在设置width和height后，再去设置把border、padding都是将内容区域变小，不会影响盒子的整体大小，盒子的整体大小还是前面设置的width和height。

  • 布局方式

    1. table布局。表格中的内容可以自动居中。阻挡浏览器渲染引擎的渲染顺序

    2. flex布局

    3. float布局（会有合并塌陷问题）

    4. 响应式布局。最简单的处理方式是加上一个 meta 标签。1 rem = 16px

  • position

    absolute：绝对定位

    相对于position不是static的父（更往上）标签来进行定位的，如果没有父标签，相对html

    fixed：固定定位

    页面向下滚动的时候，始终以浏览器窗口为参照,固定在页面上.

    relative：相对定位

    对于自己文档的位置来定位的，对旁边的元素没有影响

    static：默认值

    inherit：规定应该从父元素继承 position 属性的值
